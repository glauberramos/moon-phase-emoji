# Moon Phase Emoji

Simple api to add a emoji moon phase widget to your website

## Usage

```HTML
<iframe
  src="https://glauberramos.github.io/moon-phase-emoji/"
  height="15"
  width="15"
  style="border: none; position: fixed; right: 10px; top: 10px"
></iframe>
```
